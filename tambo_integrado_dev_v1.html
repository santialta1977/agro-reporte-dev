Escenarios guardados
Guía rápida tambo
Configurá el sistema, guardá escenarios y compará margen, sólidos y eficiencia.

Primer uso

  • Elegí cuenca y sistema productivo para cargar referencias iniciales.
  • Ajustá litros, sólidos, carga y costos a la realidad del caso.
  • Aplicá presets solo como punto de partida, no como valor final.

Escenarios

  • Guardá un escenario base antes de hacer variantes.
  • Usá nombres descriptivos para comparar precio, eficiencia o costos.
  • El historial permite recuperar escenarios guardados.

KPIs principales

El análisis prioriza margen, costo por litro, sólidos, litros por vaca y eficiencia del sistema. Revisá esas métricas antes de exportar.

Buenas prácticas

Después de guardar cambios importantes, tocá Actualizar en el dashboard principal para que KPIs y PDF usen el último escenario.

Tip: separá escenarios por hipótesis concreta, por ejemplo precio leche, dieta, sólidos o eficiencia reproductiva. Así las comparaciones son más limpias.
